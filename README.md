# WebShop Automation Framework

## Project Overview

This project is developed to automate the Demo Web Shop application using Selenium WebDriver with Java.

## Tools & Technologies

- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
- Apache POI (Excel)
- Git
- GitHub

## Framework Design

- Page Object Model (POM)
- Maven Project
- Data Driven Testing
- Utility Classes
- Base Test
- TestNG Framework

## Project Modules

- Login
- Registration
- Add To Cart
- Remove Product
- Checkout
- Logout

## Folder Structure

```
src
 ├── main
 │     ├── java
 │     └── resources
 └── test
       ├── java
       └── resources
```

## How to Execute

1. Clone the repository.
2. Import as Maven Project.
3. Update Maven Dependencies.
4. Execute testng.xml.
